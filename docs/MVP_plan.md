# MVP plan (кратко)

Цель: загрузка лабораторных документов, извлечение базовых показателей (MVP), расчёт отклонений и формирование отчёта.

## Включено в этот репозиторий

- **Backend**: Python 3.11 + FastAPI
- **DB**: PostgreSQL (Docker Compose)
- **Storage**: MinIO (S3-compatible)
- **Auth**: JWT
- **OCR**:
  - сейчас: заглушка извлечения тестов (как в исходном плане)
  - далее: подключение реального OCR (Tesseract) + парсинг текста
- **Клиенты**: скелеты web/telegram/android

## Быстрый запуск

```bash
docker compose up -d --build
```

- API: `http://localhost:8000`
- MinIO:
  - S3: `http://localhost:9000`
  - Console: `http://localhost:9001` (логин `minio`, пароль `minio12345`)

## Следующие шаги

- Реальный OCR по изображениям/PDF и извлечение показателей из текста
- Нормализация по полу/возрасту и референсам лабораторий
- Экспорт в PDF
- CI/CD, миграции Alembic, полноценный фронтенд
