# Клиенты (скелеты)

В рамках MVP этот репозиторий содержит **минимальные скелеты** клиентов:

- `frontend/web`: веб-клиент (React/Vite)
- `frontend/telegram-bot`: Telegram-бот (Python, skeleton)
- `frontend/android`: Android-клиент (Kotlin, skeleton)

Полноценная реализация UI/UX и бизнес-логики клиентов — следующий этап после стабилизации API.
